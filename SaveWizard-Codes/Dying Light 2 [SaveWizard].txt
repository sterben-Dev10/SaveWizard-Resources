Mod Skill Level: Template
800100XX VVVVVVVV
080000XX 000000NN
Notes:
XX = Value, Found in 1st column (Skill_IDS) <-- Refer to spreadsheet
V's = The ID of the skill you wish to edit
NN = The level of how high you want your skill level to be. Max FF (255)
Refer to Tested NN Values, or a custom value 1-255 (In Hex)
Use windows programming calculator 
or https://www.save-editor.com/tools/wse_hex.html
to convert Dec (Normal Number) to Hex.

Notes: Don't forget to add 0's at the end of the ID,
as the code needs 8 Bytes long on each line.
